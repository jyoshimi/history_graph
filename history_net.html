Historical Network for Psychology

First pass effort at using vis.js to display historical patterns using a visual graph.

I hope to improve this with your help! I'm not at all happy with the current look of it. Chronology should somehow be incorproated with older figures more to the left and newer more to the right. Links currenty just represent "influence" in a generic way, based on wikipedia. It would be nice to incorporate more objective links between historical figures (e.g. advisor relations, and co-citation counts) and be able to use a drop-down box to choose how these links should be displayed.

The simulation mode is just there to experiment with different graph representation styles for now. It also shows you some of the kinds of interactions avaialable to you. To get rid of it comment out the line that says "configurePhysics:true"
