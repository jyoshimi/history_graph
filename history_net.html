Historical Network for Psychology

First pass effort at using vis.js to display historical patterns using a visual graph.

I hope to improve this with your help! I'm not at all happy with the current look of it. Chronology should somehow be incorproated with older figures more to the left and newer more to the right. Links currenty just represent "influence" in a generic way, based on wikipedia. It would be nice to incorporate more objective links between historical figures (e.g. advisor relations, and co-citation counts) and be able to use a drop-down box to choose how these links should be displayed.

Todos

  • Incorporate years so that it flows roughly left to right
  • Add more fields: geography, universities, year of Phd, dates of birth and death
  • Some kind of database structure
  • Explicit link types that can be selected.
    • General influence
    • Citations
    • Advisor
    • Critic
  • Ability to associate information with links.
  • Group structure: Gestaltists, Freudians, Behavorists, Neo-Behaviorist, mathematical psyche, philosophy, social psychologists, clinicians
  • Show Images and other information
  • Click to expand
  • Also useful: The psychology family tree. Maybe able to mine this....Also see neurotree

Control panel

There is a simulation mode that allows you to experiment with different graph representation styles. To reveal or hide it comment or uncomment the line in the source that says "configurePhysics:true"
